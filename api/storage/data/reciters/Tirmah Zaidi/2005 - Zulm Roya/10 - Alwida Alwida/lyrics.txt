Sajda-E-Rab Ki Aey Zeeb-O-Zeen Alwida

Izat-E-Deen Shah-E-Mashriqeen Alwida

Aey Jamal-E-Roh-E-Qiblatein Alwida

Alwida Alwida Dil Ke Cheen Alwida

(Shaam Jati Hoon Main Aey Hussain Alwida) X3

  1. Amman Zehra Ke Aey Nor-E-Aeen Alwida

Shaam Jati Hoon Main Aey Hussain Alwida

  2. Mazher-E-Bandagi Ke Waqar Alwida

Rab-E-Kaba Ke Sajda Guzar Alwida

Sitwat-E-Deen Ke Pasdaar Alwida

Doosh-E-Khatm-U-Rusul Ke Sawar Alwida

Shaam Jati Hoon Main Aey Hussain Alwida

  3. Dasht-E-Ghurbat Ke Aey Khastatan Alwida

Waris-E-Murtaza Be Watan Alwida

Taabad Izat-E-Panj-E-Tan Alwida

Qatl Geh Se Chali Hai Behen Alwida

Shaam Jati Hoon Main Aey Hussain Alwida

  4. Sorahein Hain Kayi Gulbadan Khak Per

Ser Hain Nezoon Pe Or Un Ke Tan Khak Per

Lash-E-Serwar Bhi Hai Be Kafan Khak Per

Chhorh Ker Main Chali Bankpan Khak Per

Shaam Jati Hoon Main Aey Hussain Alwida

  5. Zam-E-Batil Ka Ser Sernigoon Hoo Gaya

Haq Pe Kamil Yaqein Ka Sila Kerbala

Aaj Be Gor Aur Be Kafan Ho To Kya

Kal Yahan Majmaye Aashiqaan Deekhna

Shaam Jati Hoon Main Aey Hussain Alwida

  6. Tum Se Dushwaar Ter Mujh Ko Manzil Mili

Be Rida Bhi Hoon Or Qayd Bhi Ho Gayi

Kufa-O-Shaam Mein Kerbla Hai Meri

Lo Bahan Imtihaan Se Guzar Ne Chali

Shaam Jati Hoon Main Aey Hussain Alwida

  7. Kerbla Mein Jo Guzri Bataoon Gi Main

Haal Maqtal Ka Saara Sunaoon Gi Main

Soorahein Hain Jo Un Ko Jagaoon Gi Main

Shaam Se Loot Ker Bhai Aaoon Gi Main

Shaam Jati Hoon Main Aey Hussain Alwida

  8. Tum Ne "Wijdan" Kesa Ye Noha Likha

Bint-E-Zehra Ki Aayi Sada Marhaba

Khuld Waloon Mein Bhi Jis Ka Chercha Hua

Jis Pe Her Gher Matam Bapa Ho Gaya

Shaam Jati Hoon Main Aey Hussain Alwida

  9. Amman Zehra Ke Aey Nor-E-Aeen Alwida

Shaam Jati Hoon Main Aey Hussain Alwida
