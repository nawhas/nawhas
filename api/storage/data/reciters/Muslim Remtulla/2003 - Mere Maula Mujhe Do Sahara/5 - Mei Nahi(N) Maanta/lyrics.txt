Jisko Shehki Gulami Muyassar Na Ho Ayse Sardar Ko, Main Nahin Manta Main Nahin
Maanta.

Jisko Ghasib Ko Munsek Ohdaa Mile, Aysi Sarkar Ko, Main Nahin Maanta Main
Nahin Maanta.

  1. Jisne A’ale Muhammed Ko Dhoka Diya, (Jisne Zehra Se Baaghe Fidak Leliya X2)

Uski Har Ek Khata Manta Huu Magar, Uska Saakar Ko, Main Nahin Maanta Main
Nahin Maanta..

  2. Jo Wasiye Rasule Khuda Se Kabhi (Apni Bai’at Ka Taalib Huwa Ho Kahin X2)

Ayse Ghaddar Ko, Ayse Makkar Ko, Ayse Ayyar Ko, Main Nahin Maanta Main Nahin
Maanta..

  3. Sacha Mo’min Hu Mei Pakka Mawlayi Hu, (Sab Sahaba Ka Aashik Hu Shezayi Hu X2)

Jo Ke Baade Nabi Aal Se Phirgaye, Ayse Do Char Ko, Main Nahin Maanta Main
Nahin Maanta..

  4. Ghasibo Ke Liye Jo Thikana Bana, (La’anate Kibriya Ka Nishana Bana X2)

Jisko Binte Mohammed Ne Radh Kardiya Ayse Darbar Ko, Main Nahin Maanta Main
Nahin Maanta..

  5. Nawha Khaane Ko Jo Geet Ka Rang De (Zikr Mawla Ki Keemat Jo Tayker Ke Le X2)

Jo Masaib Parhe Muskurate Huwe, Us Azadar Ko, Main Nahin Maanta Main Nahin
Maanta..

Maanta.
