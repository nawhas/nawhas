Lokon Main Ujar Gayin Aan (X4)

Mera Maariya Veer Hussain, Lokon Main Ujar Gayin Aan.


Zainab Paye Kardi Hai Vain (X2) Lokon Main Ujar Gayin Aan.

1)  Mazloom Hussain Di Lash Uthi, Enj Sog Di Malka Aayi Hai
     Keevai Kafan Puwawan Chan Veeran, Meri Chadar Vi Nah Ray Ay
     Khulay Vaal Tai Kardi Vain (X2 ) Lokon Main Ujar Gayi Aan.


2)  Tu Saang Uthay Main Mehmaal Tay, Hy Shaam Di Aan Nay Raavan
     Main Bhul Gayin Awn-O-Mohammad Nu, Ghazi Di An Katiyan Bavan
     Hai Kon Sunay Merai Vai’rn (X2) Lokon Main Ujar Gayin Haan.


3)  Tatheer Di Waris Sir Nangai Jadon Wich Bazar Dai Gayi Hai
     Har Pasaiyon Khalqat Saari Nai Ditay Tohfe Hasbuna De
     Mainu Ujri Zainab Kain (X2) Lokon Main Ujar Gayi Haan.


4)  Mun Ker Kai Dariya Val Zainab Akhai Veer Abbas Hun Aja
     Lut Pai Gayi Khaimai Sar Gayai Nai Sadai Perdai Aan Bacha Jaa
     Aj Kon Sunai Merai Vai’rn (X2) Lokon Main Ujar Gayi Haan.
