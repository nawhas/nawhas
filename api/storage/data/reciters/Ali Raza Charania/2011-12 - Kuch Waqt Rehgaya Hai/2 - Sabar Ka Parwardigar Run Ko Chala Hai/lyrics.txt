1- dhondti hai har ma apnay pisar ko
Kahti hai ro ro kar mola na jao
Sadma yea dil par sab say bara hai

Sabar ka parwardigar run ko chala hai

2- karbobala mai aii qayamat

Sabar ka parwardigar run ko chala hai

3- yad e padar mai roti hai har dam

‘sugra’ say janay kis nay kaha hai

Sabar ka parwardigar run ko chala hai

4- kurta jo maa nay khud say siya tha

Sabar ka parwardigar run ko chala hai

5- aun o mohd meray hawalay

Sabar ka parwardigar run ko chala hai

6- sha kay qadam say lipti hai bachi
Kahti hai baba janay na dongi

Sabar ka parwardigar run ko chala hai
